## Listening to Events

<Tabs syncKey="framework">
  <TabItem label="Vanilla JS">
```typescript
import { queryGrid } from '@toolbox-web/grid';
import type { CellClickDetail, CellCommitDetail } from '@toolbox-web/grid';

const grid = queryGrid<Employee>('#my-grid');

// Type-safe event listening
grid.addEventListener('cell-click', (e: CustomEvent<CellClickDetail<Employee>>) => {
  console.log(e.detail.row, e.detail.field);
});

// Cancelable events — call preventDefault() to reject the action
grid.addEventListener('cell-commit', (e: CustomEvent<CellCommitDetail<Employee>>) => {
  if (!isValid(e.detail.value)) {
    e.preventDefault(); // Rejects the edit
  }
});
```
  </TabItem>

```tsx
import { DataGrid, useGrid, useGridEvent } from '@toolbox-web/grid-react';
import type { CellClickDetail, CellCommitDetail } from '@toolbox-web/grid';

function EmployeeGrid() {
  const gridRef = useGrid<Employee>();

  // Type-safe event hook
  useGridEvent(gridRef, 'cell-click', (e) => {
    console.log(e.detail.row, e.detail.field);
  });

  // Or use event handler props
  const handleCellCommit = useCallback((detail: CellCommitDetail<Employee>) => {
    if (!isValid(detail.value)) {
      // Cancelable — return false or call e.preventDefault()
    }
  }, []);

  return <DataGrid ref={gridRef} rows={employees} onCellCommit={handleCellCommit} />;
}
```

## Cancelable Events

Four events support `preventDefault()` to reject the action:

| Event | Plugin | What Canceling Does |
|-------|--------|-------------------|
| `cell-commit` | Editing | Reverts the cell to its previous value |
| `row-commit` | Editing | Reverts the entire row to pre-edit state |
| `column-move` | Reorder | Rejects the column drag-and-drop |
| `row-move` | Row Reorder | Rejects the row drag/keyboard move |

:::note[Multi-Sort overrides `sort-change`]
When `MultiSortPlugin` is active, the `sort-change` event payload changes from `SortChangeDetail` (single `{ field, direction }`) to `{ sortModel: SortModel[] }` (array of sort columns). This override cannot be expressed in `DataGridEventMap` because TypeScript declaration merging doesn't allow changing a property type. Import `SortModel` from `@toolbox-web/grid/plugins/multi-sort`.
:::
